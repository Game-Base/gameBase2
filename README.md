项目结构说明：
BaseFrame：这个项目登录模块没有拆分出来，登录模块和游戏业务逻辑在一起的完整包。
BaseFrameLib：这个是从BaseFrame里拆分出来的login模块，里面包含了登录整个流程首页加载、进度条加载、登录界面加载、创角加载、配置文件加载、进入场景资源预加载、平台sdk接入模块等进入游戏所有细节的处理。
gameBase：引入BaseFrameLib登录类库的一个进入游戏的模板。
LoginLib：从BaseFrameLib类库简化过来的一个登录类库，删除了登录游戏、创角角色、预加载资源等一些功能模块，相当于一个缩减版的BaseFrameLib，用来做小游戏开发的类库。
gameBase2：这是一个引入LoginLib类库的进入游戏项目模板，此类库用来当成一个模板使用，需要新做小游戏时候整个模板拷过去直接开发，不能在此编写具体游戏业务逻辑。

以上五个模块都是作为基础模板使用，后面会持续更新优化结构
